This repository contains a simple script and documentation to help users analyze their onchain activity on the Base network.

## Features
- Track wallet transactions on Base
- Categorize activity (swaps, LP positions, mints, bridges, claims)
- Identify patterns for airdrop readiness
- Provide a clean summary of onchain behavior

## Why this project?
Base is rapidly becoming one of the most active L2 ecosystems.  
This project helps users understand their footprint on Base and optimize their onchain presence.

## Future improvements
- Add API integration (BaseScan / Blockscout)
- Add CSV import for transaction logs
- Add scoring heuristics for airdrop readiness
